# SunRaz

SunRaz is my private playground for Bootstrap.

## Features

- Using [Astro](https://astro.build/)
- Using [Bootstrap](https://getbootstrap.com/)
- Using [PurgeCSS](https://purgecss.com/)
- Customized Sass
- Using [Bootstrap Icons](https://icons.getbootstrap.com/)
